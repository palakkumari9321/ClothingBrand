# 👗 ClothingBrand

A modern full-stack clothing e-commerce website built with React.js, Node.js, Express.js, and MongoDB. This platform allows users to browse products, manage their cart, place orders, and provides an admin dashboard for product and order management.

## 🚀 Features

- User Registration & Login
- Product Search & Filtering
- Shopping Cart
- Secure Checkout Process
- Order Management
- Admin Dashboard
- Responsive Design
- REST API Integration

## 🛠️ Tech Stack

### Frontend
- React.js
- CSS
- Axios

### Backend
- Node.js
- Express.js

### Database
- MongoDB
